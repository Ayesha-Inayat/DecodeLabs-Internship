# DecodeLabs Internship - Responsive Web Design Task

This repository showcases a modern, fully responsive webpage developed as part of the DecodeLabs Internship. The project demonstrates advanced CSS techniques, fluid typography, and seamless cross-device compatibility.

## Project: Creative Studio v2

A high-performance, aesthetically pleasing landing page for a creative agency.

### Key Requirements Implemented:

1.  **CSS Media Queries:**
    *   Multiple breakpoints (`960px`, `768px`, `480px`) to handle desktops, tablets, and mobile devices.
    *   Dynamic layout shifts (e.g., flex-direction changes, grid adjustments).
2.  **Responsive Navigation:**
    *   Custom-built mobile "hamburger" menu with smooth CSS transitions.
    *   JavaScript-powered toggle functionality.
    *   Sticky header with glassmorphism effect for consistent access.
3.  **Proper Spacing and Alignment:**
    *   Use of `clamp()` for fluid typography and spacing that scales naturally.
    *   Consistent padding and margin variables.
    *   Modern Flexbox and Grid layouts for precise element positioning.

### Technologies Used:

*   **HTML5:** Semantic markup for structure and accessibility.
*   **CSS3:** Custom properties (variables), media queries, Flexbox, and Grid.
*   **JavaScript:** Vanilla JS for interactive components.

### How to Run:

1.  Navigate to the `creative-studio-v2` directory.
2.  Open `index.html` in your preferred web browser.
3.  Use the browser's developer tools to inspect the responsive behavior across various screen sizes.

# Creative Studio - Responsive Webpage

A modern, fully responsive static webpage built as part of the DecodeLabs Internship. This project demonstrates the application of fluid design principles and interactive UI components.

## Key Features

- **Fluid Architecture:** Employs the "Art of Fluidity" concept using `clamp()` for dynamic typography and relative units (rem, %, vh/vw) for a truly seamless transition between all screen sizes.
- **Responsive Navigation:** Features a robust, JavaScript-powered mobile menu with smooth CSS transitions and an interactive hamburger toggle.
- **Advanced CSS Layouts:** Utilizes Modern Grid and Flexbox with `auto-fit` and `minmax` patterns for organic element redistribution.
- **Precision Alignment:** Rigorous use of spacing variables and container constraints to ensure pixel-perfect alignment across different viewports.
- **Multi-Page Experience:** Includes dedicated 'About Us' and 'Contact Us' pages, all following the same responsive design language.
- **Modern Visuals:** Glassmorphism effects (`backdrop-filter`), custom transitions, and interactive gallery overlays.

## Technical Requirements Fulfilled

- **CSS Media Queries:** Implemented at key breakpoints (`960px`, `768px`, `480px`) to optimize the layout for various devices.
- **Responsive Navigation:** A dedicated mobile menu that activates on smaller screens, providing a clean and accessible user interface.
- **Spacing & Alignment:** Systematic use of CSS variables and modern layout techniques to maintain visual harmony and readability.

## Project Structure

- `index.html`: Main landing page with hero, services, team, testimonials, and gallery sections.
- `about.html`: Detailed information about the studio.
- `contact.html`: Contact form and location details.
- `styles.css`: Central stylesheet containing all global styles and media queries.
- `script.js`: Vanilla JavaScript for mobile menu interactivity.

## How to View

1. Open `index.html` in any modern web browser.
2. Use browser developer tools (F12) to test responsiveness by resizing the window or selecting different device emulators.

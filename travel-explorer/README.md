# Travel Explorer - Responsive Landing Page

A modern, high-conversion landing page for a travel agency, built with a focus on responsive design and user experience. This project was developed as a separate example for the DecodeLabs Internship.

## Project Overview

**Travel Explorer** is designed to showcase beautiful destinations while providing a seamless browsing experience across all device types. It utilizes a mobile-first approach for some components and robust desktop layouts for larger screens.

## Key Features & Requirements Met

### 1. CSS Media Queries
- **Responsive Breakpoints:** 
  - `960px`: Transitions to a mobile-friendly side-drawer navigation.
  - `768px`: Adjusts hero buttons and section padding for tablet-sized screens.
  - `480px`: Optimizes container padding and logo sizing for small mobile devices.
- **Fluid Layouts:** Uses Flexbox and CSS Grid with `auto-fit` to ensure content redistributes naturally.

### 2. Responsive Navigation
- **Interactive Hamburger Menu:** Custom CSS-animated toggle for mobile views.
- **Side Drawer:** A modern, slide-in navigation menu for smaller screens.
- **Sticky Header:** Stays at the top of the viewport with a subtle scroll animation for better accessibility.

### 3. Proper Spacing & Alignment
- **Fluid Typography:** Uses `clamp()` for headings to ensure they scale perfectly without breaking layouts.
- **Consistent Grid System:** A standardized 1200px container with flexible padding.
- **Visual Hierarchy:** Careful use of white space, font weights, and color contrast to guide the user's eye.

## Technical Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom variables, Flexbox, Grid, and advanced animations.
- **Vanilla JavaScript:** Handle mobile menu toggling, smooth scrolling, and scroll-triggered header effects.

## How to View
1. Open the `travel-explorer` folder.
2. Launch `index.html` in your web browser.
3. Use your browser's inspect tool (F12) to experience the responsiveness across different device simulations.

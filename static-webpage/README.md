# Creative Studio Static Webpage

A modern, clean, and responsive static webpage built using HTML5 and CSS3.

## Features
- **Semantic HTML5**: Uses proper structure with `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`.
- **Responsive Design**: Mobile-friendly layout using CSS Flexbox and Media Queries.
- **Modern UI**: Clean typography, professional color palette, and interactive hover effects.
- **Gallery**: A responsive grid of placeholder images representing a portfolio.
- **Smooth Scrolling**: Enhanced navigation experience with CSS `scroll-behavior`.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)

## How to View
1. Clone or download this project.
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).

## Project Structure
- `index.html`: Main HTML structure.
- `styles.css`: External CSS for styling.
- `README.md`: Project documentation.

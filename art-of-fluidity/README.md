# The Art of Fluidity

A showcase of responsive web design and fluid CSS techniques, now transformed into an interactive aquatic experience. This project demonstrates modern front-end development practices, focusing on fluidity, responsiveness, and aesthetic appeal using vanilla HTML, CSS, and JavaScript.

## Features

- **Interactive Water Ripple**: Real-time canvas ripple effect that responds to mouse movement and creates random "drips".
- **Aquatic Color Palette**: A refreshed visual identity inspired by ocean depths and clear water.
- **Fluid Typography**: Uses CSS `clamp()` for text that scales perfectly across devices.
- **Responsive Navigation**: A clean, modern navbar with a mobile-friendly hamburger menu.
- **Glassmorphism & Floating Cards**: UI elements that float and use backdrop filters to mimic the transparency of water.
- **Morphing Visuals**: Animated CSS shapes that evoke organic, liquid movement.
- **Semantic HTML5**: Clean and accessible document structure.

## How to View

1. Clone or download this repository.
2. Navigate to the `art-of-fluidity` directory.
3. Open `index.html` in your preferred web browser.
